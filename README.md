# Rinha de Compilers - C#

## Sobre

Projeto criado para a [Rinha de Compiladores](https://github.com/aripiprazole/rinha-de-compiler).

## Como usar

No diretório raiz, execute o comando abaixo:

```bash

docker build -t rinha-csharp .

```
Em seguida, execute:
```bash

docker run -it --rm rinha-csharp fib.json

```
